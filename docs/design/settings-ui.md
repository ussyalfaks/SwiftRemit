# Settings UI

Figma: https://www.figma.com/design/2wymodr5O54BzxyjNw8jrG/Swift-Remit-UI?node-id=33-4&t=rlRmiUDHYfdLfxvp-0

SwiftRemit Settings interface for the Stellar-based micro-remittance platform. Goal: package complex Web3 identity and security features into a clean, fintech-first mobile experience.

## Core UI/UX Contributions

- **Identity & Network Architecture** — Clear hierarchy separating personal identity from blockchain details; copy-to-clipboard for Stellar Public Key
- **Auto-donate Round-ups** — Module allowing users to contribute spare change from transactions to selected causes
- **Currency Selector** — Rapid switching between XLM and other micro-remittance assets; intuitive for non-technical users
- **Advanced Security Notifications** — Red-bordered card with shield icon indicating Multi-sig is active and core settings require secondary approval
- **Network Transparency Footer** — Persistent "Always-Synced" indicator with node version (Precision Node 0.4.2)

## Visual Language

| Aspect | Detail |
|--------|--------|
| Theme | Deep obsidian and plum palette — "Pro" feel, reduced eye strain |
| Status Badges | High-contrast "Live" and "Active" badges for immediate visual feedback |
