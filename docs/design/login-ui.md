# Login & Auth UI

Figma: https://www.figma.com/design/2wymodr5O54BzxyjNw8jrG/Swift-Remit-UI?node-id=1-5&t=rSlnndhCYxCpgNbZ-0

## 1. Login Screen

**Heading:** Welcome Back  
**Sub-heading:** Securely access your Swift Remit dashboard.

| Element | Value |
|---------|-------|
| Fields | Email or Username, Password |
| Primary Button | Login to Account → |
| Secondary | Biometric Login |
| Footer | New to Swift Remit? Create an account |

## 2. Sign Up Screen

**Heading:** Join the Future of Remittance  
**Sub-heading:** Create your decentralized account and start sending money globally in seconds.

| Element | Value |
|---------|-------|
| Fields | Email Address, Create Password, Confirm Password |
| Checkbox | I agree to the Terms of Service and Privacy Policy |
| Primary Button | Create My Account |
| Footer | Already have an account? Log In |

## 3. Forgot / Reset Password Screen

**Heading:** Create New Password  
**Sub-heading:** For your security, choose a strong password you haven't used with us before.

Password checklist:
- Minimum 8 characters
- At least one number
- At least one special character
- Mix of upper & lowercase letters

**Primary Button:** Reset Password →

## Design Decisions

- Action-oriented language ("Login to Account", "Create My Account") makes the interface feel more responsive
- "Secured with End-to-End Encryption" badge kept concise and high-contrast
- Consistent tone across all three screens: **Secure, Swift, and Simple**
