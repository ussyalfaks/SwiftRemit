# Color Palette — Web3 Design System v2.1.0-BETA

Standardizing high-signal status palettes across Web3 portfolio components to eliminate "color jitter" and create a cohesive professional UI.

Figma: https://www.figma.com/design/ajVZ2okRYUyKSojUyVRa8z/Implement-a-Consistent-Color-Palette--Standardization-?node-id=7-2&t=IjmpDaeHXfVcjCll-1

## Global Color Variables

| Token | Hex |
|-------|-----|
| Cyber-Dark | `#0A0A0A` |
| Neon Teal | `#14FFEC` |
| Purple Haze | `#B16EFF` |
| Neutral Gray | `#1A1A1A` |

## Accessibility

- 142 AAA passing combinations
- Strict WCAG 2.1 contrast compliance
- Theme-switching ready architecture

## Key Outcomes

- Reduced visual debt by 40% via centralized high-signal colors
- Eliminated 12+ redundant hex variations
- Consistent hover states across all CTAs
- `--color-accent` for primary gradients; `--color-secondary` for success-state highlights
- "Neon Teal" reserved strictly for positive growth and success states
